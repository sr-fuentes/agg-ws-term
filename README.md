## Installation

```rust
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Clone repo
git clone https://github.com/sr-fuentes/agg-ws-term.git

# Run 
cd agg-ws-term
cargo run --release
```

## Usage

`Left / Right` -> Select Asset

`Up / Down` -> Select Exchange

`q` -> Quit
